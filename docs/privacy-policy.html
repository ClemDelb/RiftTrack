Politique de confidentialité

RiftTrack — Dernière mise à jour : avril 2025

RiftTrack est une application mobile de suivi de statistiques pour League of Legends. Cette politique explique quelles données sont utilisées et comment.

1. Données collectées

RiftTrack ne collecte, ne stocke et ne transmet aucune donnée personnelle sur ses serveurs.

L'application utilise uniquement :

  • Votre Riot ID (nom de joueur + tag) saisi manuellement, stocké localement sur votre appareil.
  • Votre région de jeu, stockée localement sur votre appareil.

Ces données restent sur votre appareil et peuvent être supprimées à tout moment depuis les paramètres de l'application.

2. API Riot Games

L'application interroge directement l'API publique de Riot Games pour récupérer vos statistiques de jeu (rang, historique de parties, maîtrise des champions). Ces requêtes sont effectuées depuis votre appareil. Aucune donnée ne transite par un serveur tiers.

L'utilisation de l'API Riot Games est soumise à la politique d'utilisation de Riot Games.

3. Données de tiers

RiftTrack n'intègre aucun outil d'analyse, de publicité ou de suivi tiers (pas de Google Analytics, Firebase, Facebook SDK, etc.).

4. Données des enfants

L'application n'est pas destinée aux enfants de moins de 13 ans et ne collecte sciemment aucune donnée les concernant.

5. Sécurité

Toutes les données saisies sont stockées localement via le stockage sécurisé de votre appareil. Aucune transmission vers un serveur externe n'est effectuée par RiftTrack.

6. Vos droits

Vous pouvez supprimer toutes les données stockées par l'application à tout moment en utilisant la fonction "Supprimer le compte" dans les paramètres, ou en désinstallant l'application.

7. Contact

Pour toute question concernant cette politique de confidentialité, vous pouvez ouvrir une issue sur le dépôt GitHub : github.com/ClemDelb/RiftTrack.

RiftTrack n'est pas affilié à Riot Games. League of Legends est une marque déposée de Riot Games, Inc.
